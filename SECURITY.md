# Security Policy

This repository does not contain executable code, but we welcome responsible disclosure of any issue that may affect the integrity, privacy, or trust of this content.

## Reporting a Potential Issue
If you discover a vulnerability or content integrity issue (e.g., broken link, malicious URL, or outdated resource), please report it by creating a [security advisory](../../security/advisories/new) or by opening a private discussion with the maintainers.

We aim to review reports within **7 days** and acknowledge valid issues publicly when appropriate.

## Scope
- Link integrity
- Data authenticity
- Contributor conduct compliance

Thank you for helping maintain a trustworthy resource.
